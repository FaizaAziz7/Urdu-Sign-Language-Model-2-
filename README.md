# Urdu-Sign-Language-Model-2-

This model supports Urdu Sign Language (USL) to enable two-way communication between deaf/mute individuals and hearing people through real-time gesture recognition.

The model is trained on the Pakistan Sign Language Dataset (openPose):
https://www.kaggle.com/datasets/saadbutt321/pakistan-sign-language-dataset

The architecture combines CNN for feature extraction with LSTM layers for temporal sequence learning.

The trained model achieves 97% gesture recognition accuracy. Along with the TensorFlow Lite (.tflite) model. The model is optimized for mobile and real-time applications.
